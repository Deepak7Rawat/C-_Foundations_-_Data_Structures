## 📖 About the Repository

This repository serves as a practical codebase illustrating the journey from basic syntax and control flow to advanced topics like recursion and custom data structure implementations. Every assignment is encapsulated within its own class, ensuring clean, modular, and reusable object-oriented code.

## 🗂️ Project Index

### 1. Logic Building & Patterns
Programs focused on loops, conditionals, and mathematical series.
* **`Ass1.cpp`** - Console-based geometric pattern printing.
* **`Ass2.cpp`** - Calculates the Greatest Common Divisor (GCD) and evaluates sums/counts of odd and even digits in a number.
* **`Ass3.cpp`** - Advanced numeric grid pattern generation.
* **`Ass4.cpp`** - Interactive series summation calculator (e.g., sum of squares).
* **`Ass12.cpp`** - Numeric sequence pattern generation based on user input.

### 2. String Manipulation
Programs demonstrating character arrays, standard strings, and the `<sstream>` library.
* **`Ass5.cpp`** - Substring generation and traversal (with and without built-in string functions).
* **`Ass6.cpp`** - Word reversal within a sentence utilizing `stringstream`.
* **`Ass7.cpp`** - Advanced string manipulation and in-place reversal techniques.
* **`Ass13.cpp`** - Visual mapping of alphabetical characters based on their positional values.
* **`Ass16.cpp`** - Robust password validator (checks for uppercase, lowercase, digits, special characters, and length).
* **`Ass17.cpp`** - Extracts and sums all numerical values embedded within a given string.

### 3. Arrays & Matrices
Implementations of 1D and 2D arrays, searching, and sorting algorithms.
* **`Ass8.cpp`** - Algorithm to find the second maximum element in an array.
* **`Ass9.cpp`** - Dynamic insertion of elements into a specific index within an array.
* **`Ass10.cpp`** - Implementation of **Selection Sort** and **Bubble Sort** algorithms.
* **`Ass11.cpp`** - 2D Array operations featuring **Matrix Multiplication** with dimension validation.

### 4. Math & Recursion
Object-oriented approaches to mathematical conversions and recursive algorithms.
* **`Ass14.cpp`** - Binary to Decimal number conversion.
* **`Ass15.cpp`** - Calculates the product of all prime digits within a given number (demonstrates Class Inheritance).
* **`Ass18.cpp`** - Digit manipulation and swapping within an integer.
* **`Ass19.cpp`** - Menu-driven recursion program featuring:
  * Factorial Calculation
  * Fibonacci Series Generation
  * Tower of Hanoi Solver

### 5. Data Structures
Custom implementations of foundational linear data structures from scratch.
* **`Ass20.cpp`** - **Linear Queue** implementation featuring `Push()`, `Pop()`, and `Peek()` operations.
* **`Ass21.cpp`** - **Circular Queue** implementation for optimized memory reuse.
* **`Ass22.cpp`** - **Stack** implementation handling overflow and underflow conditions.
